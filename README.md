## 🚀 Key Features
* **Quick Diagnostic:** DNS flush + Temp files cleanup.
* **Package Manager:** Automated Winget updates.
* **System Hygiene:** Deep cleanup via `cleaner.py`.

## 🛠 Installation & Usage

### Option A: Quick Start (Recommended)
1. Click the green **Code** button above and select **Download ZIP**.
2. Extract the archive to any folder.
3. Run the tool:
python maintenance.py


### Option B: Via Terminal
git clone github.com/Ralex999/System-Toolbox.git
cd System-Toolbox
python maintenance.py
