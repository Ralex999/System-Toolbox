# 🛠 System Maintenance & Diagnostic Tool (SMDT)

A lightweight, modular utility for Windows system optimization, diagnostic reporting, and package management automation.

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)](https://www.microsoft.com/windows)

## 📖 Overview
Automated system administration tasks: network diagnostics + Winget software management.

## 🖼 Preview
![App Screenshot](System-Toolbox-main.png)

## 🚀 Key Features
* **Interactive Menu:** Simple console interface.
* **Quick Diagnostic:** DNS cache flushing + Temporary File cleanup.
* **Software Management:** Winget updates (including pinned/unknown apps).
* **System Hygiene:** Deep cleanup via `cleaner.py`.

## 🛠 Installation & Usage

1. **Python 3.8+** is required.
2. **Clone & Run:**
git clone https://github.com/Ralex999/System-Toolbox.git
cd System-Toolbox
python maintenance.py
