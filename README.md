# 🛠 System Maintenance & Diagnostic Tool (SMDT)

A lightweight utility for Windows system optimization and package management.

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/downloads/)

## 🖼 Preview

## 🚀 Key Features
* **Quick Diagnostic:** DNS flush + Temp files cleanup.
* **Package Manager:** Automated Winget updates.
* **System Hygiene:** Deep cleanup via `cleaner.py`.
